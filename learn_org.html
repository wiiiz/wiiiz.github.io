5. improvement

5.1. DONE emacs的那啥，输入法问题，在evil的插入模式自动进入之前插入模式的输入法，而在normal模式只使用英文的输入法。

就是在插入模式下的输入法模式在切换模式的时候会被记录下来，再次进入的时候恢复，而在其它模式的时候只使用英文输入法？

5.2. TODO emacs的导出可以有一个css文件来格式化导出的html，让导出更好看。

不少内容都是搜索"css for org"搜出来的。 hugo Is there a collection of CSS styles for Org exports? What are your favorites? 这个贴子里边有一个 #+HTML_HEAD: <link rel="stylesheet" type="text/css" href="https://orgmode.org/worg/style/worg.css"/> 这个好看，但是这个网页不能自动适配不同的屏幕，是不是改一改就可以了。

5.2.1. github

orgcss org-html-themes 我比较想要的格式是，目录在左边，并且可以收起来，想看的时候在展开的那种，这样比较适合手机看。 判断的逻辑是屏幕的长宽比，如果是横着的那种屏幕，那就直接显示目录。如果是竖着的目录，比如手机，那就在左上角显示一个目录图标，点击后再显示目录。具体参考arch-wiki。Arch Wiki Style
